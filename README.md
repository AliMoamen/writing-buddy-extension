## Usage

The extension will fetch the writing submission from the page and analyze the content in real-time, providing the following metrics:

- **Words:** Total word count of the submission.
- **Sentences:** Total sentence count.
- **Paragraphs:** Total paragraph count.
- **Readability Grade:** Flesch-Kincaid readability score.
- **Vocabulary Diversity:** Percentage of unique words used.
- **Transition Words Frequency:** Percentage of sentences using transition words.

You can interact with the text by editing it in the provided textarea, and the analysis will automatically update as you type. You can also preview the writing or reset it to its original state using the respective buttons.
